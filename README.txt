How to install the Pomodoro timer!
1. Open the Github to the pomodoro timer and download all the contents
2. make three files named "index.html", "main.js" and "styles.css"
3. Copy and paste the contents of the file into each of the files in terms of their names respectively
4. Make sure to import the soundeffectalarm.mp3 into the files
5. Run the program use it
